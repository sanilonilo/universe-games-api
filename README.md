# Commands:

## Run project - Docker

```shell
docker-compose up
```

## Stop project - Docker

```shell
docker-compose down
```

## Install libs

```shell
npm i
```

## Run project dev

```shell
npm run dev
```

## Run unit tests

```shell
npm run test
```

## Run coverage testing

```shell
npm run test:ci
```
